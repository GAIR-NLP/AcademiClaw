[AI Scientific Research Technology/Industry/Policy Analysis Report - Writing Instructions]

I. Report Type and Topic

- Type: Technical, industry, or policy research analysis report
- Topic: Applications of artificial intelligence technology in scientific research (applications, trends, challenges, policy/industry implications)

II. Suggested Writing Steps

1. **Determine Structure and Length**
   - Write in LaTeX, body text 4-6 pages (English)
   - Recommended sections: title, abstract, introduction, technical/application analysis, trends and challenges, policy or industry implications, conclusion, references

2. **References**
   - At least 10 reliable sources
   - Citing non-academic sources such as CSDN, Zhihu, Baidu Wenku, etc. is prohibited
   - Citation format should be standard (e.g., author, title, year, venue)

3. **Content Coverage**
   - Technical depth and correctness: AI applications in scientific research (e.g., protein folding, climate, materials, literature, etc.)
   - Scientific impact and representative applications
   - Industry trends or policy/ethics discussion

4. **Output**
   - Save the LaTeX source file as `report.tex` (or a custom main filename) in the current directory
   - If needed, compile to generate `report.pdf` and place it in the current directory as well

III. Self-Check Points

- Number of references >= 10
- No prohibited sources used (CSDN, Zhihu, etc.)
- Entire document is in LaTeX format
- Content is related to the "AI in scientific research" topic, with no large irrelevant or plagiarized sections
