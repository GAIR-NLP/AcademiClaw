# Output Format Specification  
 
你必须生成 2 个 Markdown 文件： 
- study_notes.md 
- mock_exam.md 
 
输出内容必须严格遵循下述结构与标题名称（标题必须完全一致）。 
 
--- 
 
## 1) study_notes.md 格式要求（复习笔记） 
 
文件必须包含以下一级标题（顺序固定）： 
 
# 走进极地 期末复习笔记 
 
## 0. 使用说明 
- 笔记如何使用（1段） 
- 引用格式说明（必须写出示例：文件名+页码） 
 
## 1. 全课程知识结构图（文本思维导图） 
- 用缩进/编号呈现（至少 4 层深度） 
 
## 2. 逐课件总结（必须覆盖全部课件） 
对每一份课件进行涉及知识点的总结 
 
## 3. 高频概念对照表（≥1个表） 
至少 1 个 Markdown 表格，要求对照至少 8 行概念 
 
--- 
 
## 2) mock_exam.md 格式要求（模拟试卷） 
 
文件必须包含以下一级标题（顺序固定）： 
 
# 走进极地 期末模拟试卷 
 
## 0. 说明 
- 总分100 
- 建议时长120分钟 
- 题型与分值说明 
 
## 1. 单项选择题（20题×2.5分=50分） 
每题必须用以下模板： 
 
### Q<编号> 
题干... 
A. ... 
B. ... 
C. ... 
D. ... 
【答案】<A/B/C/D> 
【解析】至少2-3句解释，说明为什么正确/错误 
【来源】<课件文件名> p.<x>-<y> 
 
## 2. 简答题（6题×5分=30分） 
模板： 
 
### Q<编号> 
问题... 
【参考答案】（建议 120-200 字） 
【评分点】（至少 3 点，每点一句） 
【来源：文件名，第 x 页】 
 
## 3. 论述题（2题×10分=20分） 
模板： 
 
### Q<编号> 
论述题... 
【参考答案】（建议 400-700 字，结构化分点） 
【评分Rubric】（至少 5 点，每点一句，含权重可选） 
【来源】至少引用 2 份不同课件（文件名+页码） 
 
--- 
 
## 通用格式要求 
- 必须使用 Markdown 
- 必须严格使用上述标题层级与名称 
- 必须保证所有“来源”字段包含：课件文件名 + 页码 
- 不得在输出中出现“我查了网络/百科/外部资料”等表述
