# AI 3602 Pilot Project — Final Project Preliminary Assignment

## Overview

This project is a preliminary assignment for the final course project. The project involves: using data mining methods combined with modern AI programming tools to mine and clean data for a **specific topic** from publicly available, relatively raw internet datasets.

- The preliminary project must be completed **individually**.
- According to the course schedule, specific requirements for **five topic directions** will be gradually released; you only need to choose one. Topics include text data (mathematics domain text), streaming data, graph data, source code, and privacy information.
- Use of data mining algorithms covered in the course is encouraged; algorithms related to data mining that are not covered in the course are also permitted.
- Use of AI IDEs, Vibe Coding, and similar tools to complete this assignment is encouraged.
- This accounts for **15%** of the final course grade.
- You are encouraged to apply the collected dataset to the final project in any manner, such as as a benchmark, training data, RAG knowledge base, etc.

## Notes

1.  **Team size limit**. The preliminary project is an **individual** project; teaming up is not allowed.
2.  **Data requirements**.
    *   Raw data should come from various webtext, common crawl, and other general internet data sources. Reference data directly related to the topic should not exceed **20%**.
    *   The use of datasets from the open-source community that have already been cleaned for existing topics is **prohibited**. Students are expected to perform data filtering and cleaning on their own.
    *   You may reference the mining and cleaning pipeline design of open-source datasets for specific topics, or use open-source datasets as supplementary material.
3.  **Methodology requirements**.
    *   Purely manual data filtering is **prohibited**. Partial manual involvement in the pipeline is allowed, but the specific workload must be noted in the report.
4.  **AI programming**.
    *   Use of AI IDEs, Vibe Coding, and similar tools to complete the project is encouraged.
