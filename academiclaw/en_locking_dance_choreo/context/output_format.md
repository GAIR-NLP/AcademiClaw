## Detailed Format Specifications

### Counts Breakdown Description Rules
1. **Beat numbers must be explicit**: Must clearly label beat ranges (e.g., 1-2, 3-4, etc.)
2. **Movements must be specific**: Use specific movement descriptions; avoid vague language
3. **Directions must be clear**: If there are direction changes, they must be explicitly stated
4. **Use technical terminology**: Use standard Locking terminology

**Example:**
```
1-2: Right foot Scooby Doo to right side, weight transfer
3-4: Left point forward with body turn, clear finger extension
5-6: Double wrist roll with shoulder isolation
7-8: Chest lock with sudden muscle contraction
```

### Locking Elements List Rules
1. **Standard names**: Use standard Locking element names
2. **Note variants**: If variants are used, note the variant type
3. **Logical ordering**: Order by appearance or importance

**Example:**
"Lock (basic), Point (forward), Scooby Doo, Wrist roll (fast variant), Stop & Go"

### Rhythm Treatment Description Rules
1. **Rhythm correspondence**: Explain how movements correspond to the musical rhythm
2. **Strong beat treatment**: Explain how strong beats are handled
3. **Syncopation**: Describe syncopation treatment in detail
4. **Rhythm layers**: Explain the rhythm layers being addressed

### Prohibited Language
1. Instructional language such as "teach the student," "teaching objective," "teaching focus"
2. Vague descriptions such as "do some movements," "roughly like this"
3. Subjective evaluations such as "cool move," "looks great"
4. Incomplete descriptions lacking necessary details

## Scoring Focus Explanation
Scoring will primarily focus on:
1. **Musical Treatment Ability** (40%): Degree of correspondence between movements and music, rhythm treatment complexity
2. **Technical Presentation Quality** (30%): Accuracy and clarity of Locking techniques
3. **Creative Choreography Level** (20%): Innovative element use and original combinations
4. **Format Compliance** (10%): Whether this format specification is followed

**Note**: Teaching-related content is not a scoring focus; the emphasis is on evaluating the quality of the choreography itself.
