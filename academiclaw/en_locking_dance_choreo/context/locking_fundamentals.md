# Locking Movement Element Library

## Core Elements (Must Master)

### 1. Lock (Freeze Point)
**Basic Technique**: Sudden stop, muscles tighten, clear position
**Common Variants**:
- **Basic Lock**: Sudden stop at any body part
- **Knee Lock**: Sudden stop with bent knees
- **Chest Lock**: Sudden stop with chest forward
- **Double Lock**: Two body parts freeze simultaneously
- **Jump Lock**: Sudden stop mid-jump
- **Slide Lock**: Sudden stop mid-slide

**Rhythm Variations**:
- On-beat Lock (1, 2, 3, 4)
- Half-beat Lock (&)
- Syncopated Lock (e, a)
- Sustained Lock (hold for 2 or more beats)

### 2. Point (Pointing)
**Basic Technique**: Arm fully extended, fingers clearly pointing
**Direction Variations**:
- Front, back, left, right
- Up, down, diagonal
- Cross-body pointing
- Sequential pointing

**Body Coordination**:
- Body rotation to match
- Eyes follow the fingers
- Footwork coordination
- Weight shift coordination

### 3. Wrist Roll
**Basic Technique**: Circular wrist rotation
**Speed Variations**:
- Slow rotation (2 beats to complete)
- Medium rotation (1 beat to complete)
- Fast rotation (half-beat to complete)
- Variable speed rotation (alternating fast and slow)

**Coordination Variations**:
- Single-hand rotation
- Alternating hands
- Synchronized double-hand rotation
- Combined with body turns

### 4. Scooby Doo
**Basic Technique**: Cross-step into glide step
**Footwork Variations**:
- Basic Scooby Doo
- Turning Scooby Doo
- Jumping Scooby Doo
- Sequential Scooby Doo

**Rhythm Treatment**:
- Scooby Doo on strong beats
- Scooby Doo on syncopation
- Rapid sequential Scooby Doo
- Slow extended Scooby Doo

### 5. Up Lock
**Basic Technique**: Arms raised upward with sudden stop
**Spatial Variations**:
- Single-arm up lock
- Double-arm up lock
- Alternating up lock
- Jumping up lock

**Combination Elements**:
- Up lock into Point
- Up lock into Lock
- Up lock into body wave
- Up lock into footwork

### 6. Which-a-way (Direction Change)
**Basic Technique**: Rapid direction switching
**Direction Combinations**:
- Front-right-back-left
- Diagonal direction changes
- Circular direction changes
- Random direction changes

**Weight Control**:
- Low center of gravity changes
- High center of gravity changes
- Rapid weight transfer
- Stable weight hold

### 7. Stop & Go
**Basic Technique**: Move-stop-move rhythm variation
**Rhythm Variations**:
- Stop 1 beat, go 1 beat
- Stop 2 beats, go 2 beats
- Stop half-beat, go half-beat
- Irregular stop & go

**Energy Variations**:
- Sudden stop, sudden start
- Slow stop, fast start
- Fast stop, slow start
- Gradual stop, gradual start

### 8. Funky Guitar
**Basic Technique**: Mimicking guitar playing motions
**Performance Variations**:
- Basic playing motion
- Exaggerated playing motion
- Combined with facial expressions
- Character/role-play elements

**Rhythm Coordination**:
- Following the guitar melody
- Matching chord changes
- Mimicking playing techniques
- Creating musical dialogue

## Rhythm Treatment Techniques

### Basic Rhythm Treatment
- **On-beat emphasis**: Clear movements on beats 1, 2, 3, 4
- **Half-beat treatment**: Quick or transitional movements on & counts
- **Syncopation**: Creating unexpected rhythms on e, a counts
- **Rhythm sustain**: Maintaining a steady base rhythm

### Advanced Rhythm Treatment
- **Multiple rhythms**: Different body parts handling different rhythms
- **Rhythm dialogue**: Movements forming a dialogue with the music
- **Rhythm variation**: Different rhythm treatments for the same movement
- **Rhythm innovation**: Creating new rhythm treatment approaches

### Syncopation Techniques
- **Basic Syncopation**: Accenting movements on off-beats
- **Double Syncopation**: Complex treatment on e& or &a
- **Rhythm displacement**: Shifting on-beat movements to off-beats
- **Rhythm layering**: Stacking multiple rhythm layers
